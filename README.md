# Bombeiro Civil Profissional

**Empresa especializada em prevenção e combate a incêndios, primeiros socorros e segurança em eventos.**

![Capa do Projeto](https://via.placeholder.com/1200x400/991b1b/ffffff?text=Bombeiro+Civil+Profissional+-+Arcoverde+PE)

## ✨ Sobre o Projeto

Site institucional moderno e tático desenvolvido para a empresa **Bombeiro Civil Profissional** de Pernambuco.

### Principais Destaques:
- Design escuro com identidade visual vermelha forte
- Equipe interativa com troca de uniformes (Vermelho → Cinza → Preto) ao passar o mouse
- Seções completas: Serviços, Equipe, Planos e Contato
- Totalmente responsivo (mobile + desktop)
- Animações fluidas

## 🛠 Tecnologias Utilizadas

- **React 19** + **TypeScript**
- **Vite** (mais rápido que Create React App)
- **Tailwind CSS v4**
- **Motion** (animações)
- **Lucide React** (ícones)

## 🚀 Como Rodar Localmente

```bash
# 1. Clone o repositório
git clone https://github.com/SEU_USUARIO/Bombeiro_Civil_FZ.git
cd Bombeiro_Civil_FZ

# 2. Instale as dependências
npm install

# 3. Rode o projeto
npm run dev
